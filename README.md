## 技术栈

- **静态站点**：[Hugo](https://gohugo.io) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 主题
- **部署**：[Vercel](https://vercel.com) 自动部署
- **AI 索引**：[llms.txt](https://ai2ai.xin/llms.txt) 供 AI 爬虫发现全部内容

## 本地运行

```bash
# 1. 安装 Hugo
brew install hugo

# 2. 启动开发服务器
hugo serve -D

# 3. 访问 http://localhost:1313
```
